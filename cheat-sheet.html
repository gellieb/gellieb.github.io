Cheat-Sheet-Ruby-BASICS

RUBY CHEAT SHEET

Integers

  • .to_i: Convert any individual string to integer
  • .even?: To return true/false if integer is even
  • first_num.downto(last_num): To count from one integer DOWN TO another(you will have to need a block of code)
  • first_num.upto(last_num)To count UP FROM one integer to another (again, you'll need a code block)

Strings

  • .to_s: Convert anything to a string
  • .downcase: To lowercase all letters
  • .upcase: To put in all caps
  • .capitalize: To capitalize just the first word
  • .chars: or .split(""): To split the string into individual string-letters
  • .length: To get the number of characters in the string
  • .include?(some_char): To return true/false whether some character/letter is included in the string
  • .concat: To concatenate strings

Arrays

  • .any?: To return true/false if any element within the array passes the condition stated in the block
  • .each: To iterate (go through) each element in the array
  • .map(!): Similar to "each", but permanently changes the original array
  • .include?: To return true/false if a particular element(x) is included in the array
  • .pop: To pop out (permanently delete) the last element in the array
  • .push: To add (permanently) to the array
  • .empty?: To return true/false if an array has no elements
  • .each_slice(3) {|x| p x}: To group the elements in an array (ie. groups of 3)

Hashes

  • Hash[array]: To convert an (even numbered) array into a hash
  • delete(key): To delete a key-value pair
  • .keys: To return all of the keys
  • .values To return all of the values
  • .value?(value): To check if a hash has a certain value
